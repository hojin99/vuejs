# vuejs

## 환경

* 개발툴 - Visual Studio Code 사용

* vue cli를 통한 프로젝트 생성  
https://cli.vuejs.org/guide/  
vue cli 설치  
`npm install -g @vue/cli`  
vue cli 업데이트  
`npm update -g @vue/cli`  
vue 프로젝트 생성(상위 디렉토리에서 실행, 기본 구성)  
`vue create vuejs`

* vue-router, vuex 사용  

* element-ui, axios 사용  


## 실행  
package.json의 scripts를 통해서 실행  
serve - 웹브라우저를 띄워주면, 수정 시 자동 반영 됨
